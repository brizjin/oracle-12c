   Многопоточная компиляция операций.

   Максимальное количество сессий компиляции определяется количеством процессоров 
   на сервере Oracle.

   - Запускать файл start_compile.bat
     В качестве параметров скрипта укажите строку входа на схему и количество 
     потоков компиляции. Формат запуска файла start_compile.bat:
     {ИМЯ_ВЛАДЕЛЬЦА}/{ПАРОЛЬ_ВЛАДЕЛЬЦА}@{ИМЯ_СХЕМЫ} <количество потоков компиляции>
      Например:
         start_compile ibs/pasw@ibso 4
      Для контроля выполнения компиляции рекомендуется запустить монитор 
      коммуникационного канала с именем канала DEBUG$100.
      В  алгоритме  компиляции  предусмотрено  12  стадий,  признаком успешного
      выполнения скрипта надо считать вывод в канал DEBUG$100 строк:
         **********************************************************
         Finished
         Перекомпиляция всех методов
