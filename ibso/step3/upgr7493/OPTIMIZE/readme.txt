В каталоге OPTIMIZE находятся скрипты по оптимизации Технологического Ядра.

Скрипты, предназначенные для запуска вручную:

1) REVERSE_IDX_OSH_OBJ_ID 
    Скрипт пересоздает (создает в случае отсутствия) индекс IDX_IBS_OSH_OBJ_ID в качестве реверсивного.
    Индекс по OBJ_ID в IBS_OSH должен быть реверсивным – только так удается справиться с конкуренцией за ITL. 
    Требуется ТЯ версии выше чем 7.4.2.3.
    Запускается из-под схемы ревизора (AUD).
 
2) SEGMENT_TABLES
    Cкрипты segment_tables предназначены для оптимизации кластерной конфигурации, и расчитаны на количество 
    нод в кластере <=4. Если в инсталляции нод больше, то необходимо либо руками внести корректировку 
    в segment_tables.sql в блок: 
	partition by list (instance)
	(
	partition inst_0 values (0),
	partition inst_1 values (1),
	partition inst_2 values (2),
	partition inst_3 values (3)
	);
    ,либо запросить ее у ЦФТ.
    Скрипт рекомендуется запускать в тех случаях, когда в топах конкуренция за таблицы 
    FRM_LOCKS, RTL_USERS, RTL_LOCKS и индексные блоки на них.
    Установка обновления ТЯ в режиме upgrade откатывает изменения данного скрипта.
    Запускается из-под схемы владельца.
