Инструкция по установке лицензионной информации

Лицензионная информация состоит из следующих файлов:

1. Файл со списком уполномоченных лиц. 
Файл представляет собой подписанный X.509 сертификатом документ в формате
xml, c расширением *.lic.

Файл выдает Менеджер Банка.

2. Файл лицензии. 
Файл представляет собой подписанный X.509 сертификатом документ в формате
xml, c расширением *.lic.

Файл выдает Менеджер Банка.

3. Файл описания приложения.
Файл представляет собой подписанный X.509 сертификатом документ в формате
xml, c расширением *.xsap.
  Например, для приложения ЦФТ-Банк должно быть 2 файла с описанием:
  Файл с описанием приложения "ЦФТ-Платформа Развития" и файл с описанием
  приложения "ЦФТ-Банк".

Файл с описанием приложения "ЦФТ-Платформа Развития" версии 7.4 лежит в текущей папке: Core_7.4.*.xsap
Файл с описанием приложения "ЦФТ-Банк" взять на сайте сопровождения в разделе ЦФТ-Банк в разделе обновления.
Файл с описанием приложения "ЦФТ-Ритейл Банк" взять на сайте сопровождения в разделе ЦФТ-Ритейл 
Банк в разделе обновления.

	
Порядок установки лицензионной информации:

1. Запустить АРМ "Администратор дистрибутива" версии 6.19.0.32 или
выше. Зайти на схему с правами владельца схемы.

2. Вызвать окно "Лицензирование" по команде меню "Управление" - "Лицензирование".

3. На панели слева содержатся команды для работы с лицензионной
информацией. Запустить мастер установки.

4. В окне "Выбор установки" мастера установки выбрать "Установка только описаний приложений".

5. В окне "Задание путей к описаниям приложений" нажать "Выбрать..." и выбрать
core*.xsap в корневой папке upgrade. Нажать кнопку "Далее >".

6. В окне "Установка лицензии" нажать кнопку "Готово". Выбранное описание будет установлено в Систему.
